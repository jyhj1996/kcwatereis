<하단 조회 컬럼 금월>

목표량 : 목표 등록 시 해당 거래처의 품목 수량 합계(거래처품목별 목표등록(KCW))

목표액 : 목표 등록 시 해당 거래처의 품목 금액 합계(거래처품목별 목표등록(KCW))

목표단가 : 목표액 / 목표량(소수점 반올림하여 표기)

실적량 : 출하 & 마감 수량 데이터 합계

달성률 : 실적량 / 목표량(백분율로 표기, 소수점 둘째자리까지 표현)

실적단가 : 실적액 / 실적량(소수점 반올림하여 표기)

단가차이 : 실적단가 - 목표단가

실적액 : 출하 & 마감 금액 데이터 합계

달성률 : 실적액 / 목표액(백분율로 표기, 소수점 둘째자리까지 표현)

잔여수량 : 목표량 - 실적량

잔여금액 : 목표금액 - 실적금액

예상매출액 : (조회기간 매출액 / 조회기간 영업일수) x (금월 말일까지 잔여 영업일수) (* 조회기간이 월을 넘어 설 경우 해당 그리드 제외)

예상달성률 : (예상매출액 / 목표액) * 100 (백분율로 표기, 소수점 둘째자리까지 표현) (* 예상매출액이 존재 할 경우만 표기하며 아닌 경우 예상매출액과 같이 그리드 제외)

<하단 조회 컬럼 전월대비>

실적량 : 출하 & 마감 수량 데이터 합계

실적액 : 출하 & 마감 금액 데이터 합계

단가차이 : 전월실적단가 - 당월실적단가

실적량 증감률 : (금월 실적량 - 전월 실적량) / 전월 실적량 * 100

실적액 증감률 : (금월 실적량 - 전월 실적량) / 전월 실적액 * 100

※ 1개월 조회 시 나타나지만 그 이상 조회 시 사라짐.(전월대비)

ex) [조회기간] 2023/01/01 ~ 2023/01/31 조회 경우

금월 : 2023/01/01 ~ 2023/01/31 기간

전월대비 : 2022/12/01~2022/12/31 기간

전년대비 : 2022/01/01 ~ 2022/01/31 기간

[조회기간] 2023/01/01 ~ 2023/05/31 조회 경우

금월 : 2023/01/01 ~ 2023/05/31 기간

전월 : Null 값(전원대비 컬럼 사라짐.)

전년대비 : 2022/01/01 ~ 2022/05/31 기간

<하단 조회 컬럼 전년대비>

실적량 : 출하 & 마감 수량 데이터 합계

실적액 : 출하 & 마감 금액 데이터 합계

단가차이 : 전년실적단가 - 당월실적단가

실적량 증감률 : (금월 실적량 - 전년 실적량) / 전년 실적량 * 100 (* 백분율로 표기, 소수점 둘째자리까지 표현)

실적액 증감률 : (금월 실적액 - 전년 실적액) / 전년 실적액 * 100 (* 백분율로 표기, 소수점 둘째자리까지 표현)
